Description


The Population Genomics Viewer (PGV) is a custom genome browser designed to provide the user with reliable and detailed answers to questions on the haplotype diversity and phylogenetic origin of the genetic variation underlying any genomic region.

PGV allows the user to select a region of the genome and a set of samples. The region is selected by specifying the start (e.g. 31200000 or 31200K or 31.2M), and end of the interval and the chromosome (i.e, autosome number and X chromosome). Samples can be selected by name or by entire set.

Data sets


SNPs

This display provides local density of SNPs. The browser provides a histogram in which the height of the bar is proportional to the local SNP density.
